TaoGetVariableBoundVecs

 Get the vectors with the lower and upper bounds in current solver.

Synopsis

#include "tao.h"  
int TaoGetVariableBoundVecs(TAO_SOLVER tao, Vec *XL, Vec *XU)

Input Parameters

tao -the TAO solver

Output Parameter

XL - the lower bounds
XU - the upper bounds

Note

These vectors should not be destroyed.

Keywords

Application, bounds
