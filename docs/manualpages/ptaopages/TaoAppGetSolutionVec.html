TaoAppGetSolutionVec

Synopsis

#include "tao.h"  
int TaoAppGetSolutionVec(TAO_APPLICATION taoapp, Vec *X)

Input Parameters

taoapp -the application

Output Parameter

X -the solution vector

Note
