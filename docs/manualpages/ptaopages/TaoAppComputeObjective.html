TaoAppComputeObjective

#include "tao.h"  
int TaoAppComputeObjective(TAO_APPLICATION taoapp, Vec X, double *f)

Input Parameters

taopp - the TAO_APPLICATION context
X - the point where the objective should be evaluated

Output Parameter

f -function value
