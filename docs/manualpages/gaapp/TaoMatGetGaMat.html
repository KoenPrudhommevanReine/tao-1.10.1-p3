TaoMatGetGaMat

 If the TaoMat is of the TaoMatGa type, this routine gets the underlying GA matrix.

Synopsis

int TaoMatGetGaMat (TaoMat * MM, GAMat * M)

Input Parameter

MM -the TaoMat

Output Parameter

M -the GA mat

Note

The function TaoMatGa::GetMat() will also return the Mat M.

Level:advanced
