TaoCreateProjectedLineSearch

 Create a line search

Synopsis

#include "tao_solver.h" 
int TaoCreateProjectedLineSearch(TAO_SOLVER tao)

Input Parameters

tao -TAO_SOLVER context

Note

This routine is used within the following TAO bound constrained minimization solvers: TRON (tao_tron) and limited memory variable metric (tao_blmvm). This line search projects points onto the feasible, bound constrained region. It only enforces the Armijo descent condition.

TAO_SOLVER, linesearch
