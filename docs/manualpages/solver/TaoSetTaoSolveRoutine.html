TaoSetTaoSolveRoutine

 Sets the routine that will set the options for the optimization solver.

Synopsis

#include "tao_solver.h"  
int TaoSetTaoOptionsRoutine(TAO_SOLVER tao, 
                          int (*options)(TAO_SOLVER,void*))
Collective on TAO_SOLVER

Input Parameters

solver - the TAO_SOLVER context
options - routine that views the solver

Note

This routine is generally used within a "TaoCreate_XXX" routine. TAO will call this routine as part of the TaoSetOptions() routine.

Note

When TAO calls these routines, the second argument will be the context specified in TaoSetTaoSolveRoutine().

Keywords

TAO_SOLVER, solve

See Also

TaoCreate(), TaoSetOptions(), TaoSetTaoSolveRoutine().
