TaoSetDefaultParameters

 Set the parameters used by all TAO solvers to a default value. These parameter include convergence tolerances. This routine is called before setting the method used by TAO

Synopsis

#include "tao_solver.h"  
int TaoSetDefaultParameters(TAO_SOLVER solver)
Collective on TAO_SOLVER

Input Parameters

solver -the TAO_SOLVER solver context

Keywords

options, defaults

See Also

TaoCreate(), TaoSetDefaultStatistics(), TaoSetDefaultMonitors()
