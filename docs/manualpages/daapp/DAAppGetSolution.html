DAAppGetSolution

 Sets a pointer to an existing vector that contains the solution.

Synopsis

#include "taodaapplication.h" 
int DAAppGetSolution(TAO_APPLICATION daapplication, PetscInt level, Vec *X)
Collective on TAO_APPLICATION

Input Parameters

daapplication - the DA Application object
level - the DA on which the current solution is desired (Level 0 is coarsest)

Output Parameters

X -the current solution

See Also

DAAppSetBeforeMonitor(), DAAppSetAfterMonitor();

Keywords

Solution, DA
